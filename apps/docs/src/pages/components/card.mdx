# {frontmatter.title}

Cards contain content and actions about a single subject.

## Basic usage

The `Card` comes with two variants: filled, outlined.

<Playground>
  <BasicExample client:load />
  <div slot="code">
    ```tsx
    <Card class="p-5">Some content</Card>
    <Card variant="outlined" class="p-5">
      Some content
    </Card>
    ```
  </div>
</Playground>

## Composition

Add buttons to `Card`:

<Playground>
  <CompositionExample client:load />
  <div slot="code">
    ```tsx
    <Card class="p-5">
      Some content
      <div class="mt-2 flex items-center gap-x-2 text-right">
        <Button variant="text">Cancel</Button>
        <Button variant="outlined">Cool</Button>
      </div>
    </Card>
    ```
  </div>
</Playground>
