# {frontmatter.title}

Icon buttons are commonly found in app bars and toolbars.

## Basic usage

The `IconButton` comes with four variants: filled, tonal, outlined and standard.

<Playground>
  <BasicExample client:load />
  <div slot="code">
    ```tsx
    <IconButton>
      <Heart />
    </IconButton>
    <IconButton variant="tonal">
      <Heart />
    </IconButton>
    <IconButton variant="outlined">
      <Heart />
    </IconButton>
    <IconButton variant="standard">
      <Heart />
    </IconButton>
    ```
  </div>
</Playground>

## States

### Disabled

You can disable the `IconButton` using the `isDisabled` prop:

<Playground>
  <DisabledExample client:load />
  <div slot="code">
    ```tsx
    <IconButton isDisabled>
      <Heart />
    </IconButton>
    <IconButton variant="tonal" isDisabled>
      <Heart />
    </IconButton>
    <IconButton variant="outlined" isDisabled>
      <Heart />
    </IconButton>
    <IconButton variant="standard" isDisabled>
      <Heart />
    </IconButton>
    ```
  </div>
</Playground>
