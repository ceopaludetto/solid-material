# {frontmatter.title}

A Floating Action Button (FAB) performs the primary, or most common, action on a screen.

## Basic usage

The `FAB` comes with three variants: surface, secondary and tertiary.

<Playground>
  <BasicExample client:load />
  <div slot="code">
    ```tsx
    <FAB icon={<Plus />} />
    <FAB icon={<Plus />} variant="secondary" />
    <FAB icon={<Plus />} variant="tertiary" />
    ```
  </div>
</Playground>

## Extended mode

When you add a `children`, `FAB` will behave in extended mode:

<Playground>
  <ExtendedExample client:load />
  <div slot="code">
    ```tsx
    <FAB icon={<Plus />}>Add</FAB>
    ```
  </div>
</Playground>

_Note: extended mode intentionally does not respect `size` prop_

## Sizes

Also `FAB` can accept a `size` prop:

<Playground>
  <SizeExample client:load />
  <div slot="code">
    ```tsx
    <FAB icon={<Plus />} size="small" />
    <FAB icon={<Plus />} size="medium" />
    <FAB icon={<Plus />} size="large" />
    ```
  </div>
</Playground>
